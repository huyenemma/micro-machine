Tell whether or not the system supports \mbox{\hyperlink{classsf_1_1Vulkan}{Vulkan}}. 

This function should always be called before using the \mbox{\hyperlink{classsf_1_1Vulkan}{Vulkan}} features. If it returns false, then any attempt to use \mbox{\hyperlink{classsf_1_1Vulkan}{Vulkan}} will fail.

If only compute is required, set {\itshape require\+Graphics} to false to skip checking for the extensions necessary for graphics rendering.
