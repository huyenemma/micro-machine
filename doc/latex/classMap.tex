\doxysubsection{Detailed Description}
Manages the game map\textquotesingle{}s graphical representation. 

The \mbox{\hyperlink{classMap}{Map}} class is responsible for handling the visual aspects of the game map. It loads the map texture, handles its scaling, and provides functionality for rendering the map on the game window.
