\doxysubsection{Detailed Description}
Abstract base class for objects that can be drawn to a render target. 

\mbox{\hyperlink{classsf_1_1Drawable}{sf\+::\+Drawable}} is a very simple base class that allows objects of derived classes to be drawn to a \mbox{\hyperlink{classsf_1_1RenderTarget}{sf\+::\+Render\+Target}}.

All you have to do in your derived class is to override the draw virtual function.

Note that inheriting from \mbox{\hyperlink{classsf_1_1Drawable}{sf\+::\+Drawable}} is not mandatory, but it allows this nice syntax \char`\"{}window.\+draw(object)\char`\"{} rather than \char`\"{}object.\+draw(window)\char`\"{}, which is more consistent with other SFML classes.
