\doxysubsection{Detailed Description}
Manages the physics world and game entities like vehicles, collectables, and obstacles. 

The \mbox{\hyperlink{classWorld}{World}} class is responsible for updating the physics world, keeping track of game entities, and determining the winner of the game.
