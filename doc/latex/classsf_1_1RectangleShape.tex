\doxysubsection{Detailed Description}
Specialized shape representing a rectangle. 

This class inherits all the functions of \mbox{\hyperlink{classsf_1_1Transformable}{sf\+::\+Transformable}} (position, rotation, scale, bounds, ...) as well as the functions of \mbox{\hyperlink{classsf_1_1Shape}{sf\+::\+Shape}} (outline, color, texture, ...).
