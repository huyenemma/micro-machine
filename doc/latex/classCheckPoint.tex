\doxysubsection{Detailed Description}
Represents a checkpoint in the game world. 

The \mbox{\hyperlink{classCheckPoint}{Check\+Point}} class is used to create checkpoints in the game world. \mbox{\hyperlink{classVehicle}{Vehicle}} need to visit all hidden checkpoints to consider as finishing one round
