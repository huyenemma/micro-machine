Activate or deactivate the context as the current target for rendering. 

A context is active only on the current thread, if you want to make it active on another thread you have to deactivate it on the previous thread first if it was active. Only one context can be active on a thread at a time, thus the context previously active (if any) automatically gets deactivated.
