\doxysubsection{Detailed Description}
Manages game resources including images, fonts, and sounds. 

The \mbox{\hyperlink{classResourceManager}{Resource\+Manager}} class is responsible for loading, storing, and providing access to various game resources. It uses an internal mapping to link resource keys to their corresponding loaded resources.
