\doxysubsection{Detailed Description}
Manages the main game menu interface. 

The \mbox{\hyperlink{classGameMenu}{Game\+Menu}} class extends the \mbox{\hyperlink{classBaseMenu}{Base\+Menu}} class to provide a specific interface for the main game menu. It includes options for starting a single player or two-\/player game, or exiting the game.
