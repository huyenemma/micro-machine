\doxysubsection{Detailed Description}
Manages the main aspects of the game including rendering, updating, and game state management. 

The \mbox{\hyperlink{classGame}{Game}} class handles the main game loop, event processing, and switching between different states of the game. It is responsible for initializing and managing game resources, rendering game elements, and updating game logic.
