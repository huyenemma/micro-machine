Provide a string representation for a given scancode. 

The returned string is a short, non-\/technical description of the key represented with the given scancode. Most effectively used in user interfaces, as the description for the key takes the users keyboard layout into consideration.

\begin{DoxyWarning}{Warning}
The result is OS-\/dependent\+: for example, \mbox{\hyperlink{structsf_1_1Keyboard_1_1Scan_aa42fbf6954d6f81f7606e566c7abe875a5e1b12c4476475396c6f04eccbbf04f7}{sf\+::\+Keyboard\+::\+Scan\+::\+LSystem}} is \char`\"{}\+Left Meta\char`\"{} on Linux, \char`\"{}\+Left Windows\char`\"{} on Windows and \char`\"{}\+Left Command\char`\"{} on mac\+OS.
\end{DoxyWarning}
The current keyboard layout set by the operating system is used to interpret the scancode\+: for example, \mbox{\hyperlink{classsf_1_1Keyboard_acb4cacd7cc5802dec45724cf3314a142ab50635b9c913837d1bd4453eec7cb506}{sf\+::\+Keyboard\+::\+Semicolon}} is mapped to \char`\"{};\char`\"{} for layout and to \char`\"{}Ã©\char`\"{} for others.
