A class to manage and display the winning screen of the game. 

\mbox{\hyperlink{classWinnerBoard}{Winner\+Board}} is a subclass of \mbox{\hyperlink{classBaseMenu}{Base\+Menu}}. It manages the display of the winner\textquotesingle{}s information and provides menu options for replaying the game or exiting.
