Detailed Description

Here we redefine some methods to allow grabbing fullscreen events.

◆ acceptsFirstResponder

- (BOOL) acceptsFirstResponder

Allow to grab fullscreen events.

acceptsFirstResponder and canBecomeKeyWindow messages must return YES to grab fullscreen events.

See https://stackoverflow.com/questions/999464/fullscreen-key-down-actions

Returns
YES

◆ canBecomeMainWindow

- (BOOL) canBecomeMainWindow

Allow fullscreen windows to become the main window.

By default, fullscreen windows (using NSBorderlessWindowMask) cannot become the main window, so this method is changed to always return YES.

Returns
YES
